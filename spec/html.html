# 语法

  • 缩进为 2 个空格；
  • 在属性上，使用双引号，不要使用单引号；
  • 属性名全小写；
  • img、input 缩写，缩写结尾请空一格；
  • 不要忽略可选的关闭标签；
<!DOCTYPE html>
<html>
  <head>
    <title>Page title</title>
  </head>
  <body>
    <img src="..." alt="img" />
    <h1 class="hello-world">Hello, world!</h1>
  </body>
</html>

# HTML5 doctype

  • 在页面开头使用这个简单地 doctype 来启用标准模式，使其在每个浏览器中尽可能一致的展现；
  • 虽然 doctype 不区分大小写，但是按照惯例，doctype 大写
  <!DOCTYPE html>
  <html>
    ...
  </html>

# lang 属性

  • 应在 html 标签上加上 lang 属性。这会给语音工具和翻译工具帮助，告诉它们应当怎么去发音和翻译。
  • 语言列表
  • 默认使用 zh-cn
  • 字符编码指定为'UTF-8'。
<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
  </head>
  ...
</html>

# 引入 CSS，JS

  • 根据 HTML5 规范，通常在引入 CSS 和 JS 时不需要指明 type，因为 text/css 和 text/javascript 分别是他们的默认值。
<!-- External CSS -->
<link rel="stylesheet" href="...">

<!-- In-document CSS -->
<style>
    ...
</style>

<!-- External JS -->
<script src="..."></script>

<!-- In-document JS -->
<script>
    ...
</script>
