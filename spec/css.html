  • 缩进：缩进为 2 个空格.

  • 分号：每个属性声明末尾都要加分号.

  • !important：尽量不要使用!important.

  • 引号：

    • 最外层统一使用双引号
    • url 的内容要用引号
    • 属性选择器中的属性值需要引号
.element:after {
  content: "";
  background-image: url("logo.png");
}

input[type="text"] {
  ...
}
